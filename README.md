# Novio

Novio is a premium online tech courses platform designed to empower learners worldwide with industry-expert instruction. It features a modern, responsive storefront with course discovery, filtering, cart management, and user account features.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- SEO-focused markup (JSON-LD, Open Graph, Twitter Cards)

## Key Features

- 100+ premium tech courses across Web Development, Data Science, AI, Cybersecurity, Cloud Computing, and more.
- Responsive design with mobile-first approach.
- Course filtering by category, level, and price with real-time search.
- Navbar search with live suggestions and course images.
- Shopping cart with localStorage persistence.
- User authentication flow (login, signup, forgot password).
- Infinite horizontal scrolling testimonials.
- Premium footer with Coursera-style link organization.
- Engraved company logo section featuring Google, Microsoft, Meta, Amazon, IBM, NVIDIA, and more.
- Full SEO: canonical URLs, Open Graph, Twitter Cards, JSON-LD on every page, sitemap.xml, robots.txt.

## Pages

- **index.html** - Homepage with hero, trusted companies, featured courses, testimonials
- **courses.html** - Full course catalog with filtering and sorting
- **course-detail.html** - Individual course detail page
- **about.html** - About the platform and team
- **contact.html** - Contact form
- **blog.html** - Blog with articles on tech education
- **careers.html** - Career opportunities
- **cart.html** - Shopping cart
- **payment.html** - Checkout flow
- **login.html / signup.html / forgot-password.html** - Authentication
- **account.html** - User account management
- **privacy-policy.html / terms-of-service.html / refund-policy.html** - Legal pages
- **accessibility.html** - Accessibility statement
- **affiliates.html** - Affiliate program

## Getting Started

1. Clone the repository.
2. Open `index.html` in a browser or serve with any static server.
3. No build step required.
